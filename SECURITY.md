# Security

## Scope

This repository is a bounded public demo repository. It is not a production system and provides no security guarantee.

The local demo gateway is included only to simulate the bounded public demo contract.

## Reporting

Do not submit real secrets, credentials, regulated or sensitive business data, private evidence, or private runtime logs.

If you discover sensitive material that should not be present in this repository, please contact the repository owner directly.

## Status

Public visibility, release, deployment, tagging, production use, compliance claims, security claims, sandbox claims, isolation claims, and external platform integration claims remain outside this repository's demo scope.
