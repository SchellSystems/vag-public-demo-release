# Release and Operational Documents

This directory contains dated review, decision, and operational records for the public bounded demo repository.

## Reading Rule

A dated document is a historical snapshot unless this index explicitly says otherwise. A recorded SHA identifies the state that was reviewed; it is not a live pointer to the current `main` head.

Current repository facts—visibility, default branch, head SHA, open pull requests, workflow state, releases, tags, packages, and deployments—must be checked live on GitHub.

These documents do not authorize a release, tag, deploy, package publication, visibility change, or any claim beyond the bounded repository status defined in the root README and Claims and Non-Claims document.
