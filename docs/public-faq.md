# Public FAQ

## Is this production-ready?

No. This is a bounded public demo repository, not a production system, compliance proof, security product, sandbox, or enterprise platform.

## Is this a sandbox?

No. This public demo does not claim to be a sandbox. It does not provide OS-level, browser-level, network-level, process-level, or filesystem isolation.

## Does Deny prove system-wide non-execution?

No. The gateway rejects commit for a denied proposal, and the UI derives that no local artifact, Commit, or Verify followed in its bounded public-demo path. This repository contains no ToolGrant subsystem. The result does not prove that execution was blocked at the OS, browser, network, process, filesystem, or any other system level.

## Does Verify approve actions?

No. Verify checks stored hash, signature, and proposal/decision reference relationships within scope. It does not approve, authorize, or certify safety.

## Does Evidence prove compliance?

No. Evidence records and reconstructs a bounded demo path while distinguishing gateway records from UI-derived statements. It does not prove compliance, certify security, or represent full telemetry.

## Is this the VAG Core?

No. This repository contains no canonical VAG Core implementation. It contains a minimal local demo gateway that simulates the bounded public demo contract.

## Does this repository contain private Gateway code?

No. This repository contains no private Gateway or canonical Core implementation code. It contains a minimal local demo gateway for the bounded public demo only.

## Does this repository contain real Evidence?

No. All evidence examples in this repository are synthetic and conceptual. They are marked with `synthetic: true`, `conceptual_example: true`, `runtime_replayable: false`, and `not_real_evidence: true`.

## Is this a public release?

This repository is a bounded public demo repository. No production release, deploy, or production visibility claim is made by this repository.

## Can this be used for real production workflows?

No. This is a bounded local demonstration. The UI creates a local demo artifact, and the gateway receives only its caller-supplied digest. No real production workflow or external platform integration exists in this repository.

## Is this externally integrated?

No. This repository contains no production system integration. Any workflow examples are synthetic and limited to the bounded demo concept.
