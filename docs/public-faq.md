# Public FAQ

## Is this production-ready?

No. This is a private quarantine repository for curated public-demo preparation. It is not production-ready, not deployed, and not released.

## Is this a sandbox?

No. VAG does not claim to be a sandbox. It does not provide OS-level, browser-level, network-level, process-level, or filesystem isolation.

## Does Deny prove system-wide non-execution?

No. Deny stops the bounded Pilot path from producing ToolGrant, Commit, and Verify artifacts. It does not prove that execution was blocked at the OS, browser, network, process, or filesystem level.

## Does Verify approve actions?

No. Verify checks hash, signature, and reference integrity within scope. It does not approve, authorize, or certify safety.

## Does Evidence prove compliance?

No. Evidence records and reconstructs a bounded demo path. It does not prove compliance, certify security, or represent full telemetry.

## Is this the VAG Core?

No. The VAG Core is in a distinct private repository (VAG-v1-main). This repository contains only rewritten and synthetic public-demo preparation material.

## Does this repository contain private Gateway code?

No. This repository contains no Gateway implementation code. It contains only documentation and synthetic examples.

## Does this repository contain real Evidence?

No. All evidence examples in this repository are synthetic. They are marked with `synthetic: true` and `not_real_evidence: true`.

## Is this a public release?

No. This repository is private. No publication, release, deploy, or visibility change has been performed.

## Can this be used for real production workflows?

No. This is a bounded local demonstration. No real production workflow or external platform integration exists in this repository.

## Is this externally integrated?

No. This repository contains no production system integration. Any workflow examples are synthetic and limited to the bounded demo concept.
